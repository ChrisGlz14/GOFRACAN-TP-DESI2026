# GOFRACAN-TP-DESI2026

Este proyecto es un Trabajo Práctico desarrollado para la materia DESI (2026). Consiste en una aplicación web que sigue el patrón de diseño **MVC (Modelo-Vista-Controlador)**.

## 🛠️ Tecnologías y Herramientas Utilizadas

* **Lenguaje:** Java 17
* **Framework:** Spring Boot
* **Arquitectura:** MVC (Model-View-Controller)
* **Gestor de Dependencias:** Apache Maven
* **IDE Recomendado:** Eclipse IDE

## 🚀 Requisitos Previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:
* [Java Development Kit (JDK) 11](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)
* [Eclipse IDE](https://www.eclipse.org/downloads/) (con soporte para Spring Boot / Spring Tools Suite)
* [Apache Maven](https://maven.apache.org/) (o utilizar el Maven embebido en Eclipse)
